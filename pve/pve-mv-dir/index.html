pve 移动默认目录

2022-11-29
阅读时长: 1 分钟

以 移动local节点的目录为例，先配置分区自动挂载到pve 查看本文 https://dev.leiyanhui.com/pve/auto-mount-exfat

我这里主要是移动images目录，也就是虚拟机的lxc的目录
关闭所有虚拟机和lxc，

1
2

cd /var/lib/vz/
mv images /nvme/pve_vm_images # mkdir /nvme/pve_vm_images

pve 会自动新建image目录，所以我们要同时执行rm 和ln

1

rm -rf images &&  ln -s /nvme/pve_vm_images images

注意新装的pve 有可能是没有images目录的 完毕

template 目录 不建议直接移动，毕竟要用的iso的和lxc镜像并不多，直接在对应目录下ln -s 可能更简单一些。

如果要处理 一样 执行

1
2
3
4

cd /var/lib/vz/template
#mv iso /exfat/iso
rm -rf iso &&  ln -s /exfat/iso  iso
rm -rf iso &&  ln -s /exfat/iso  iso

其他的目录也可以修改 dump private snippets

/var/lib/vz/dump

1
2
3
4
5
6

cd /var/lib/vz
mv dump /exfat/dump-pve-back
rm -rf dump &&  ln -s /exfat/dump-pve-back  dump

mv snippets  /exfat/exfat/snippets-pve
rm -rf snippets &&  ln -s /exfat/snippets-pve  snippets
