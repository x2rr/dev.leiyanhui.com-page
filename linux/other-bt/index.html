再见宝塔，关于宝塔的17个替代品

2022-12-07
阅读时长: 1 分钟

宝塔经历过几次严重漏洞，加上最近小bug不断，甚至有的bug反反复复，如同月经一样，每个月出现一次（宝塔周更）

还有之前直播间的事故，强制绑定手机号等等….官网bug论坛的机器人一样的回复：我没有bug，你更新一下

反正就是死不承认，然后一直到反馈的人多了，甚至过去数周才去修复，然后过几周可能又复现

这个曾经优秀的国产软件，也可能是因为用户群太多，也可能是内部管理真的是混乱不堪。频繁爆出bug 还有 安全问题。

去年开始，新机器上就避免再使用宝塔。

这次宝塔的操作 也非常惊人，宝塔官方自己的demo都被黑掉，依旧不承认有漏洞，然后在论坛开始疯狂删反馈贴！

搞笑的是，宝塔自己的demo 迟迟都没有处理掉，第二天早上直接挂403了。 真的是。。。

首先说，还是推荐尽量手搓的方式来管理生产服务器，真的不建议再用宝塔这种鬼东西。

如果实在是需要图形界面 或者一键环境 可以看一下相关替代产品

替代品

1、https://cockpit-project.org/

Cockpit 是由 Red Hat 开发的使得管理服务器变得轻松简单的自由开源软件。它非常适合于进行简单的系统管理任务和新手系统管理员。

2、https://github.com/midoks/mdserver-web UI跟宝塔差不多，但个人试用了一下发现占用资源挺高的。

3、https://hestiacp.com/ 太简单，功能不足

4、https://oneinstack.com/ 类lnmp，无web UI

5、https://cyberpanel.net/ 太简单，功能不足

6、https://lnmp.org/ 无web UI

7、https://github.com/DevHaoZi/Panel 耗子面板，仅支持redhat系列

8、http://amh.sh/ amh面板

9、https://www.appnode.com/

10、https://vestacp.com/

11、https://fastpanel.direct/

12、https://www.urlos.com/

13、https://www.hws.com/

14、https://www.digitalocean.com/community/tools/nginx?global.app.lang=zhCN

15、 Nginx Proxy Manager https://nginxproxymanager.com/ 纯nginx反代 和 le证书管理工具

16、 纯证书管理 https://github.com/acmesh-official/acme.sh

17、超过25年的老古董Webmin/Virtualmin
